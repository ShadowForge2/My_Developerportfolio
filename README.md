# Bashir Abdulganiyy - Personal Portfolio Website

A modern, responsive personal portfolio website built with pure HTML, CSS, and minimal JavaScript.

## Features

- **Dark Theme**: Modern dark design with neon blue accent color (#38bdf8)
- **Fully Responsive**: Optimized for both mobile and desktop devices
- **Smooth Animations**: Scroll-triggered animations, hover effects, and typing effect
- **Interactive Elements**: 
  - Fixed navigation with smooth scrolling
  - Animated skill progress bars
  - Counter animations for statistics
  - Interactive contact form (UI only)
  - Project cards with hover effects

## Sections

1. **Navigation**: Fixed top navbar with smooth scroll to sections
2. **Hero**: Eye-catching introduction with typing animation
3. **About**: Professional introduction with profile image
4. **Skills**: Animated progress bars showing technical proficiency
5. **Stats**: Animated counters for achievements
6. **Projects**: Grid layout of featured projects
7. **Contact**: Contact form and information
8. **Footer**: Copyright and social media links

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS with custom properties, Grid, Flexbox
- **JavaScript**: Vanilla JS for interactions and animations
- **Google Fonts**: Inter font family

## Design Elements

- **Color Scheme**:
  - Primary Background: #0f172a (dark slate)
  - Secondary Background: #1e293b (slate)
  - Accent Color: #38bdf8 (neon blue)
  - Text Primary: #f1f5f9 (light slate)
  - Text Secondary: #94a3b8 (muted slate)

- **Typography**: Inter font family for clean, modern look
- **Animations**: Fade-in effects, hover states, parallax scrolling
- **Layout**: CSS Grid and Flexbox for responsive design

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Or run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

### Profile Information
- Update name, title, and description in the HTML file
- Replace placeholder images with actual photos
- Modify contact information

### Colors
- Edit CSS custom properties in `:root` section of `styles.css`
- Adjust accent colors to match personal brand

### Content
- Add/remove projects in the projects section
- Update skills and proficiency levels
- Modify statistics in the stats section
